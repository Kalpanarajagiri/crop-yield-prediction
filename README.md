# Crop Yield Prediction Using Machine Learning

## Project Description

This project predicts crop yield using Machine Learning.

Users enter:

- Crop Name
- Season
- State
- Land Area
- Annual Rainfall
- Fertilizer
- Pesticide
- Average Temperature

The application predicts the estimated crop yield using a Random Forest Regressor model.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
